# Run locally

1. Clone this repo

2. Navigate to that folder
   `cd <folder_name>`

3. install dependencies
   `npm install`

4 . run application
`npm run dev`

5. Endpoints for 1,2 and 3 are

```
/ -> for multistep form
/list -> for list
/e-comm-products -> for e commerce page


```
